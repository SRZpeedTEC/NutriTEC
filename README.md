# NutriTEC
NutriTec es una aplicación web y movil diseñada hacia la gestión nutricional de usuarios por medio de registros diarios, metas, gestión de productos y seguimientos personalizados. Basada en el ecosistema C# para el backend, con bases de datos en SQL Server y MongoDB y una arquitectura frontend basada en JavaScript, CSS, HTML en React diseña con IA.
